# 📖 Monkey Pox Tracker <a name="about-project"></a>

  The Monkey pox tracker is a website that displays the cases and deaths caused by monkey pox for different countries around the world.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Nodejs
- Node Package Manager
- VS Code text editor

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/ikennarichard/mpox-tracker.git
```

### Install

Install this project with:

  ```bash
    cd my-project
    pnpm install
  ```

### Usage

To run the project, execute the following command:
  ```bash
    pnpm dev
  ```
